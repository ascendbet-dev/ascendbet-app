ASCENDBET SEASON 0 PRODUCT REQUIREMENTS DOCUMENT
Product Overview

AscendBet is a performance-based betting discipline challenge where users participate in a 21-day virtual betting evaluation using a ₦100,000 starting balance.

The goal is to grow the balance to ₦120,000 without breaching a hard drawdown floor of ₦88,000.

Season Rules

Season duration:

21 days

Starting balance:

₦100,000

Target balance:

₦120,000

Hard Drawdown Floor:

₦88,000

Qualification Requirements

User must:

Reach ₦120,000

Avoid balance ≤ ₦88,000

Place ≥ 20 settled bets

Be active ≥ 8 betting days

Place max 3 bets per day

Place only pre-match bets

Stake Rules

Stake per bet must be:

₦500 – ₦2,000
(0.5% – 2% of starting balance)

Odds Rules

Singles:

Minimum odds = 1.80

Accumulators:

2–5 legs

Each leg odds must be:

1.30 – 2.50

Total ticket odds must be:

1.80 – 6.00

Drawdown Logic

User is disqualified ONLY IF:

current balance ≤ ₦88,000
AND

user has no pending bets

Season Window

User may NOT select a match that ends beyond the season end date.

Active Day Logic

A betting day counts as active only if user places at least one bet on that day.

Bet Settlement

All bet settlements must be automatically resolved via API-Football FT match result.

Backend Enforcement

All business logic must run in backend.

Frontend must NOT enforce:

stake rules

odds rules

DD logic

qualification logic
